¿Este documento tiene cláusulas opcionales (ELECCION USUARIO)?

Si el Word contiene bloques marcados con ELECCION USUARIO al inicio y al final, el sistema preguntará al cliente si desea incluir o excluir cada cláusula. Para cada cláusula que el cliente incluya, puedes definir campos adicionales personalizables.
